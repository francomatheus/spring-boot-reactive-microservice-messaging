# spring-boot-reactive-microservice-messaging


### Objective
- This project was created to trainning estructure about messaging using some services from AWS.

- Stacks: 
  - Spring boot
  - Kotlin
  - WebFlux
  - Mongo
  - AWS
  - Prometheus
  - Grafana
  - Jaeger
  - Terraform

- Projects: 
  - ConfigServer
  - Gateway
  - Discovery
  - Producer
  - Consumer
  - SNS
  - SQS
  - Lambda

- Archtecture
  - Hexagonal
